Projects

iama.bike

Description: I wrote this project in an afternoon to practice Object Oriented JavaScript. I believe I completed that goal very well because the code is very modular and reusable. I implemented a custom renderer and animation loop in vanilla JavaScript.
Challenges: Since I wanted to implement this with pure, vanilla JavaScript, drawing the different parts of the scene was difficult. Finding all the curves/points by hand was a tedious process, which in hindsight could have been sped up by using svg files.

EZ Care

Description: For the course COGS 187a at UCSD, groups of students had to brainstorm, design, and create a prototype for a new app or website. My group, EZ Care (previously EZ Mode and/or EZ Life), created an app with a smart watch geared towards elderly people. We worked together each week to complete deliverables including meeting minutes, a weekly blog, and design sketches to name a few.
Challenges: Working in a team of 7 people proved challenging at times. Not everyone had the same ideas about what the app/smartwatch should be, however we were able to work together to create an even better product concept.
